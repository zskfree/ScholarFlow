ScholarFlow

一个基于 AI 的文献综述自动化工具

// 核心模块 (new_workflow)

项目采用模块化设计，支持多种 LLM 后端与处理流程：

  • app.py: 基于 Flask 的 Web 交互界面，提供可视化进度与配置管理。
  • workflow.py: 命令行工作流入口，适合批量自动化处理。
  • pdf_to_markdown.py: 集成 MarkItDown 与 LLM 视觉能力，支持 OCR 识别。
  • llm_client.py: 统一封装 Gemini, GLM-4, OpenAI 等多模型调用。
  • summary_generator.py: 核心摘要生成逻辑，支持自定义 Prompt。
  • reference_matcher.py: 智能匹配 PDF 内容与参考文献列表。

// 如何运行

请确保已安装 Python 3.12+，并按照以下步骤操作：

# 1. 安装依赖
pip install -r requirements.txt

# 2. 配置环境
#    - 复制模板：cp new_workflow/config\ example.yaml new_workflow/config.yaml
#    - 在 config.yaml 中填入您的 API Key (Gemini/Zhipu/OpenAI)

# 3. 准备输入
#    - PDF 文件放入 new_workflow/pdfs/
#    - 编辑研究主题与参考文献列表 (.txt)

# 4. 启动 (推荐使用 Web 界面)
python new_workflow/app.py
# 访问 http://127.0.0.1:5000
